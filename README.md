# 드럼비트 연습기 🥁

메트로놈, 드럼 패턴 연습, 루딤먼트 훈련, 템포 트레이너를 제공하는 드럼 연습 도구입니다.

## 주요 기능

- ✅ 메트로놈 (BPM 40-220)
- ✅ 기본 드럼 비트 패턴 재생 (킥, 스네어, 하이햇)
- ✅ 난이도 모드 (Beginner / Intermediate / Advanced)
- ✅ 연습 모드 (Play-along / 뮤트 모드)
- ✅ 오늘의 드럼 비트 (매일 새로운 패턴)
- ✅ 루딤먼트 훈련 (Paradiddle, Single/Double Stroke 등)
- ✅ 템포 트레이너 (점점 빨라지는 BPM 연습)
- ✅ 박자 선택 (4/4, 3/4, 6/8)
- ✅ 모바일 최적화
- ✅ SEO 최적화

## 사용 방법

### 기본 비트 모드
1. 난이도 선택 (Beginner / Intermediate / Advanced)
2. 박자 선택 (4/4, 3/4, 6/8)
3. BPM 슬라이더로 템포 조절
4. 재생 버튼으로 드럼 패턴 듣기
5. 뮤트 모드로 메트로놈만 들으며 연습

### 루딤먼트 모드
1. 루딤먼트 패턴 선택
2. BPM 조절
3. 재생 버튼으로 R/L 표시 따라 연습

### 템포 트레이너 모드
1. 시작 BPM, 목표 BPM, 연습 시간 설정
2. BPM 증가량 설정
3. 시작 버튼으로 점점 빨라지는 템포 연습

## 기술 스택

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- Web Audio API

## 배포

GitHub Pages 또는 정적 호스팅 서비스에 배포할 수 있습니다.

### GitHub Pages 배포
1. GitHub 저장소 생성
2. 파일 업로드 또는 Git으로 Push
3. Settings → Pages에서 GitHub Pages 활성화
4. Source를 `main` 브랜치로 설정

### OG 이미지 및 메타 태그 설정
`index.html` 파일에서 다음 URL을 실제 배포 URL로 변경하세요:

```html
<meta property="og:url" content="https://yourusername.github.io/drumbeat/">
<meta property="og:image" content="https://yourusername.github.io/drumbeat/og-image.png">
```
